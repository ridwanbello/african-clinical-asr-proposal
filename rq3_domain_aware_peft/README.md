# RQ3: Domain-Aware PEFT

## Research Question

What domain-aware adaptation strategy can best balance specialization to African-accented clinical speech with preservation of general-domain ASR capability?

## Goal

Develop and evaluate a domain-aware PEFT strategy, such as selective-layer adaptation or adaptive rank allocation, to improve clinical-domain ASR while preserving general-domain performance.

## Expected Outputs

- Proposed method description
- Preliminary evidence table
- Expected evaluation plan
- RQ3 methodology for proposal
