# RQ1: Error Analysis

## Research Question

What error patterns do state-of-the-art ASR models exhibit on African-accented clinical speech, and how do these patterns vary across clinical and general-domain speech?

## Goal

Evaluate pretrained ASR models on AfriSpeech-200 clinical and general-domain speech, then analyze WER, CER, substitution, deletion, insertion, and qualitative error patterns.

## Expected Outputs

- Baseline WER/CER table
- Clinical vs general-domain comparison
- Error type breakdown
- Qualitative error examples
- RQ1 methodology and preliminary results for proposal
