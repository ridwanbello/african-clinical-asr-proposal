# RQ2: PEFT vs Full Fine-Tuning

## Research Question

How do parameter-efficient fine-tuning methods compare with full fine-tuning for adapting Whisper-based ASR models to African-accented clinical speech?

## Goal

Compare full fine-tuning, LoRA, DoRA, and AdaLoRA for adapting Whisper models to African-accented clinical ASR.

## Expected Outputs

- PEFT comparison table
- Clinical WER/CER results
- Trainable parameter comparison
- Efficiency and general-domain preservation table
- RQ2 methodology and preliminary results for proposal
